# MiniNote

一个 macOS 上的轻量级纯文本编辑器，体验对标 Windows 11 新版记事本。

---

## 为什么做这个

我从 Windows 换到 Mac，很多东西都适应了，但一直想念 Windows 11 新版记事本。

不是因为它多强大，恰恰是因为它够简单。打开就写，关了不丢，粘贴过来就是干净文字。没有格式，没有富文本，没有"智能"提示。就是一个安静的写字的地方。

macOS 上找了很久，要么太重（Obsidian、Typora），要么太简陋（系统自带的文本编辑），要么要订阅。没有一个刚好够用的。

所以自己做了一个。

## 它能干什么

**打开就写，关了不丢。** 随便开个标签页写东西，关机、重启、断电，内容都在。不用操心保存，不用命名文件。想起来了再保存，不想保存就关掉。

**无格式粘贴。** 从网页、微信、PDF 复制过来的文字，粘贴进去就是干净的纯文本。再复制出去，格式全没了。我经常拿它当格式清理器用。

**Finder 右键新建。** 任意文件夹里右键，直接新建一个文档。不用先打开软件再另存为。

**Markdown 可选渲染。** 可以在里面写 Markdown，也可以按 `Cmd+R` 切换渲染预览。但默认它就是一个纯文本编辑器，不做多余的事。三种文件类型（.mint / .txt / .md）在设置中有独立的渲染开关，你说了算。

**轻。** 启动秒开，常驻后台不到 50MB。没有云同步，没有账号系统，没有后台服务。就是一个工具，用完放下。

## 界面

```
+-------------------------------------------+
| 菜单栏                                     |
+-------------------------------------------+
| [未标题]  [notes.mint]  [ideas.md]  [+]   |
+-------------------------------------------+
|                                           |
|              文档编辑区域                  |
|                                           |
+-------------------------------------------+
| 状态栏（行/列 | 字符数 | UTF-8 | LF | 文本）|
+-------------------------------------------+
```

## 快捷键

| 功能 | 快捷键 |
|------|--------|
| 新建 | `Cmd+N` |
| 打开 | `Cmd+O` |
| 保存 | `Cmd+S` |
| 另存为 | `Cmd+Shift+S` |
| 关闭标签页 | `Cmd+W` |
| 撤销 / 重做 | `Cmd+Z` / `Cmd+Shift+Z` |
| 查找 | `Cmd+F` |
| 查找替换 | `Cmd+Option+F` |
| Markdown 渲染切换 | `Cmd+R` |
| 偏好设置 | `Cmd+,` |

## 支持的格式

| 格式 | 说明 |
|------|------|
| .mint | MiniNote 专属格式，默认新建。纯文本 + 轻量状态信息（光标位置等） |
| .txt | 标准纯文本，兼容其他编辑器 |
| .md | Markdown 格式 |

三种格式本质都是纯文本，互相转换就是改个后缀的事。

## 安装

从 [GitHub Releases](https://github.com/vivalucas/mininote/releases) 下载最新版本：

- `MiniNote-[版本号].dmg` — 拖拽到 Applications 文件夹
- `MiniNote-[版本号].zip` — 解压即用

应用未签名，首次打开请在文件上右键 → 打开，或到系统设置中允许。

## 系统要求

macOS 26 Tahoe 及以上，Apple Silicon / Intel。

## 开发

技术栈：Swift 6 + SwiftUI + NSTextView（TextKit），零第三方依赖。

详细设计文档见 [project-log/](project-log/)。

```bash
git clone <repo-url>
open MiniNote.xcodeproj
# Cmd+B 构建
```
