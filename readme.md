MiniNote

一个 macOS 上的纯文本小工具。



为什么做这个

我从 Windows 换到 Mac，很多东西都适应了，但一直想念 Windows 11 新版记事本。


不是因为它多强大，恰恰是因为它够简单。打开就写，关了不丢，粘贴过来就是干净文字。没有格式，没有富文本，没有"智能"提示。就是一个安静的写字的地方。


macOS 上找了很久，要么太重（Obsidian、Typora），要么太简陋（系统自带的文本编辑），要么要订阅。没有一个刚好够用的。


所以自己做了一个。



它能干什么

打开就写，关了不丢


随便开个标签页写东西，关机、重启、断电，内容都在。不用操心保存，不用命名文件。想起来了再保存，不想保存就关掉。


无格式粘贴


从网页、微信、PDF 复制过来的文字，粘贴进去就是干净的纯文本。再复制出去，格式全没了。我自己经常拿它当格式清理器用。


Finder 右键新建


任意文件夹里右键，直接新建一个文档。不用先打开软件再另存为。


Markdown 支持


你可以在里面写 Markdown，也可以选择渲染预览。但默认它就是一个纯文本编辑器，不做多余的事。.md 文件打开时会问你要不要渲染，你说了算。


轻


启动秒开，常驻后台几乎不占资源。没有云同步，没有账号系统，没有后台服务。就是一个工具，用完放下。



支持的格式

格式	说明
.mint	MiniNote 自己的格式，默认新建
.txt	标准纯文本
.md	Markdown

三种格式本质都是纯文本，互相转换就是改个后缀的事。
